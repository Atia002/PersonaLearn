PersonaLearn
Tagline: Learn your way, at your pace.

PersonaLearn is a personalized adaptive learning platform.
It is not a generic LMS and not just an AI chatbot.
The interface must communicate that every learner gets a different learning path, study plan, explanation style, examples, quizzes, and recommendations based on their:

goals
available time
pace
learning preferences
interests and hobbies
confidence
performance history
weak topics
uploaded class materials

The design must feel like a premium modern AI-first education product similar in quality to top SaaS products, not a student project template.

Create a complete desktop-first responsive design system and full product flow, including:

landing page
authentication flow
onboarding flow
diagnostic assessment flow
personalized dashboard
learning session pages
AI tutor interface
upload materials flow
quiz and feedback flow
weekly planner
profile/settings
instructor analytics
admin content manager

The final result must look exceptionally polished, interactive, professional, and premium.
